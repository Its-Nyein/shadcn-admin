# Credits

## Original Repository

This project is a modified version of the original [Shadcn Admin Dashboard](https://github.com/satnaing/shadcn-admin) created by [@satnaing](https://github.com/satnaing).

## Original Project Details

- **Repository**: [satnaing/shadcn-admin](https://github.com/satnaing/shadcn-admin)
- **Original Author**: [@satnaing](https://github.com/satnaing)
- **Original Tech Stack**:
  - Built with **Vite** and **React.js**
  - UI: ShadcnUI (TailwindCSS + RadixUI)
  - Routing: TanStack Router
  - TypeScript

## Modifications

This version has been adapted to use **Next.js** instead of the original React.js/Vite setup, as Next.js is preferred for this implementation. The core UI components, design patterns, and overall structure remain inspired by the original work.
