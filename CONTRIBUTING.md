## How to Contribute üëª

We welcome contributions in the form of bug reports, feature requests, code improvements, and more. Here's how you can contribute:

1. **Fork the Repository:** Click the ["Fork"](https://github.com/Its-Nyein/shadcn-admin/fork) button on the top right of the repository's page.

2. **Clone Your Fork to Your Local Machine:**

   ```bash
   git clone https://github.com/Its-Nyein/shadcn-admin.git
   ```

   Or if you've forked it, clone your fork:

   ```bash
   git clone https://github.com/yourusername/shadcn-admin.git
   ```

3. **Create a New Branch:**

   ```bash
   git checkout -b your-branch-name
   ```

4. **Make Your Changes and Commit Them:**

   ```bash
   git add .
   git commit -m 'Commit message'
   ```

5. **Push Your Changes to Your Fork:**

   ```bash
   git push origin your-branch-name
   ```

6. **Create a Pull Request:**

   Go back to the repository on GitHub and click the ["New Pull Request"](https://github.com/Its-Nyein/shadcn-admin/pulls) button on your branch. Fill out the form with a title and a detailed description, then submit the pull request.

7. **Wait for Your Pull Request to Be Reviewed:**

   We will review your pull request and suggest any necessary changes. Once everything looks good, we will merge your pull request. üéâ

8. **Sync Your Fork (Optional):**

   If the original repository has been updated and you would like to sync your fork with it, you can do so by adding the original repository as a remote and pulling from it:

   ```bash
   git remote add upstream https://github.com/Its-Nyein/shadcn-admin.git
   git pull upstream main
   ```
